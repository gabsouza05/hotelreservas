Hotel Reservas

Este sistema foi desenvolvido para realizar o gerenciamento de quartos e reservas de um hotel.


Funcionalidades

  • Cadastrar quartos.
  • Listar quartos cadastrados.
  • Excluir quartos.
  • Cadastrar reservas.
  • Visualizar reservas.
  • Excluir reservas.

Tecnologias Utilizadas

  • HTML
  • CSS
  • JavaScript
  • Node.js
  • Express
  • MySQL
